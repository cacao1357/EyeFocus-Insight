S10: EPA (Eye Activity) 分析报告
日期: 2026-05-30

一、分析方法

EPA (Eye Pixel Area) 传统上指虹膜在图像中的像素面积。
本分析使用 MediaPipe Blendshapes 作为眼动活跃度的代理指标。

注：真正的 EPA 需要计算关键点围成的像素面积，当前方案使用 blendshapes 系数作为替代。

二、眼部 Blendshapes 统计

| 指标 | 不戴眼镜 | 戴眼镜 | 比率 |
|------|---------|--------|------|
| eyeBlinkLeft | 0.0080 | 0.0420 | 5.3x |
| eyeBlinkRight | 0.0048 | 0.0149 | 3.1x |
| eyeWideLeft | 0.0907 | 0.0186 | -4.9x |
| eyeWideRight | 0.1120 | 0.0500 | -2.2x |
| eyeSquintLeft | 0.0610 | 0.2050 | 3.4x |
| eyeSquintRight | 0.0316 | 0.1340 | 4.2x |

三、关键发现

1. **眨眼频率**：戴眼镜时 eyeBlink 显著更高（5.3x / 3.1x）
   - 可能原因：镜片导致眼睛干燥，触发更多眨眼

2. **眼睛睁开程度**：不戴眼镜时 eyeWide 显著更高（4.9x / 2.2x）
   - 可能原因：眼镜框架轻微压迫或镜片缩小了视觉感知

3. **眯眼程度**：戴眼镜时 eyeSquint 显著更高（3.4x / 4.2x）
   - 可能原因：镜片导致轻微视觉变形，习惯性眯眼补偿

四、眼动活跃度指数

计算公式：blink_left + blink_right + wide_left + wide_right

| 条件 | 活跃度指数 |
|------|-----------|
| 不戴眼镜 | 0.2155 |
| 戴眼镜 | 0.1255 |
| 差异 | 0.0900 |

结论：不戴眼镜时眼动更活跃（+72%）

五、眯眼比率（潜在疲劳指标）

计算公式：squint / (squint + wide)

| 条件 | 左眼眯眼比率 | 右眼眯眼比率 |
|------|-------------|-------------|
| 不戴眼镜 | 0.40 | 0.22 |
| 戴眼镜 | 0.91 | 0.73 |

结论：戴眼镜时眯眼比率显著更高，可作为：
1. 眼镜检测的辅助特征
2. 疲劳检测中区分眼镜用户的影响因素

六、与 Phase 0 S5 结论的对比

Phase 0 S5 发现：EAR 方差法无法区分眼镜（眼镜反而降低方差）

新发现：Blendshapes 可以有效区分眼镜状态：
- 眯眼比率差异显著（0.40 vs 0.91）
- 眨眼频率差异显著（5.3x）
- 眼睛睁开程度差异显著（4.9x）

七、结论

1. Blendshapes 可作为眼镜检测的有效特征
2. 眯眼比率 (squint / (squint + wide)) 是最具区分度的指标
3. 眨眼频率可作为眼镜检测的辅助指标
4. Phase 1 建议：结合眯眼比率 + 眨眼频率 + eyeWide 综合判断眼镜状态

八、后续建议

1. 采集更多样本（每次条件 5-10 次）建立统计显著性
2. 探索眼角关键点距离（方案 B）作为补充特征
3. 考虑用 MLP 模型融合多特征进行眼镜分类
