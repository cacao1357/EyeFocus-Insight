## 五、风险与缓解

| 风险 | 概率 | 影响 | 等级 | 缓解措施 |
|------|:--:|:--:|:--:|---------|
| PyInstaller MediaPipe 打包失败 | 中 | 高 | 🔴 | Spike 先行验证 mediapipe 能否被 PyInstaller 正确打包；备选：提供 .bat 启动脚本 + Python 安装指南 |
| PyQt5 打包后 plugins 路径错误 | 中 | 高 | 🔴 | spec 文件显式指定 `QT_QPA_PLATFORM_PLUGIN_PATH` |
| sklearn/scipy 打包体积过大 | 高 | 中 | 🟡 | 接受 250-350MB；提供 lite 版本预案（仅 OpenCV + Qt） |
| 无 GPU 机器 FPS 不达标 | 中 | 高 | 🟡 | CPU 模式下降低分辨率（640→480）；确保 XNNPACK 推理 < 15ms |
| 无摄像头环境测试 | 中 | 中 | 🟡 | exe 启动时检测摄像头，无摄像头时显示提示并退出 |
| 文档/PPT 与代码不一致 | 中 | 中 | 🟡 | 文档随代码改；最终验收前做一遍全流程截图 |
